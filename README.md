# Manish Kunthoor — Full-Stack Developer Portfolio

Welcome to my personal portfolio repository. This project showcases my skills, credentials, education, and real-world web applications. It has been redesigned from the ground up to feature a premium, high-impact glassmorphic developer interface.

**Live Preview:** [my-site-newpage.vercel.app](https://my-site-newpage.vercel.app/)

---

## 🎨 Design Overhaul Features

This portfolio is crafted with the latest design paradigms to provide a responsive, premium user experience:

- **Orbital Backdrop Blobs:** Dynamic CSS animated radial-gradient background nodes orbiting fluidly in the background.
- **Floating Island Navigation:** A centered, glassmorphic pill-shaped navigation header that tracks scroll sections and highlights the active route automatically.
- **Mock Developer Terminal:** An interactive mock editor presenting developer attributes in standard JSON structure.
- **Dynamic Skill Illumination:** Interactive tile panels representing technologies that illuminate their borders and shadow casts in the official brand color (e.g. React blue, Node green) upon hover.
- **Vertical Interactive Timeline:** A clean history path for educational milestones containing glowing connection nodes.
- **Project Feature Docks:** Re-architected project presentations showing features in clear checklist form with separate live and source repositories.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18.x or higher)
- npm or yarn package manager

### Local Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jayeshgith/Portfolio_mani.git
   cd Portfolio_mani/portfolio
   ```

2. **Install node dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the local application.

### Production Build

```bash
# Build the optimized production output
npm run build

# Start the built application server
npm run start
```
